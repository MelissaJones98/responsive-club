# Project: The Witch's Book Nook - Book Club Website

## Description:
As part of a Codecademy course I was tasked to create a website for a club, I chose to create a website for a fictional book club, showcasing skills in responsive web design.

## Features:
The website contains:
- a home page that greets the user and introduces the book club
- a brief history of the club creation and details of the founder
- details about upcoming and previous books that the club has read
- a schedule of the upcoming club meetings as well as details of the meeting location
- a sign up page for people to become members of the book club

## How to Use:
Run the index.html file in the browser and navigate the site using the buttons on the nav bar. 

## Technologies:
- HTML
- CSS
